# QIES Kyber WASM Module

Version: 0.1.0
Build timestamp: 20250530-144301

## Usage

To run the example:

1. Start a web server in this directory:
   ```
   python -m http.server 9000
   ```

2. Open a web browser and navigate to:
   ```
   http://localhost:9000/kyber.html
   ```

## Files

- `qies_kyber_bg.wasm`: WebAssembly binary
- `qies_kyber.js`: JavaScript bindings
- `qies_kyber.d.ts`: TypeScript definitions
- `kyber.html`: Example HTML file
